Adding AUTHOREA to existing repository

When you aim to add AUTHOREA to an existing directory you can add a new document in AUTHOREA using the name of the repository as the title of the document. Then setup automatic GitHub integration and the AUTHOREA files will be added as a separate branch in the corresponding repository on GitHub. Just make sure that the AUTHOREA files wont conflict with existing files in the repository and merge to AUTHOREAbranch into the master
